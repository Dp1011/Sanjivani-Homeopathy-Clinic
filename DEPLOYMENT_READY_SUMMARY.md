## 🚀 Quick Start Command

When you're ready to deploy, this is all you need on your production server:

```bash
# Clone and setup
git clone https://github.com/Dp1011/Sanjivani-Homeopathy-Clinic.git homeopathy-clinic
cd homeopathy-clinic
npm install

# Configure
cp .env.example .env
nano .env  # Add your settings

# Build
npm run build

# Run
npm install -g pm2
pm2 start npm --name "homeopathy-clinic" -- start
pm2 startup
pm2 save
```

That's it! Your site is live!
