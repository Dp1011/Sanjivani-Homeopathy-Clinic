## 🎨 Design System Details

### **Pattern Used: Trust & Authority + Social Proof**
Perfect for healthcare providers. Includes:
- ✅ Visible credentials and qualifications
- ✅ Years of experience highlighted
- ✅ Patient testimonials with ratings
- ✅ Professional certifications
- ✅ Trust-building statistics

### **Style: Soft UI Evolution**
- Soft shadows: `shadow-sm hover:shadow-md`
- Subtle depth via gradients
- Calming color palette (greens + cream)
- Premium feel with gold accents
- Professional serif typography

### **Animations**
- **Duration**: 300ms (calming speed)
- **Type**: Smooth transitions on hover
- **Effects**: Subtle scale, shadow lift, color shifts
- **No jarring animations** - respects UX principles

## 📞 Support & Customization

The design system is **fully customizable**:

### To Modify Colors:
Edit `tailwind.config.ts`:
```typescript
colors: {
  clinic: {
    sage: "#3d5a45",      // Change primary color
    moss: "#5c7a63",      // Change secondary
    accent: "#c4a574",    // Change accent
    // etc...
  }
}
```

### To Add/Edit Testimonials:
Edit `src/app/page.tsx`:
```typescript
const testimonials = [
  {
    name: "Patient Name",
    condition: "Health Condition",
    rating: 5,
    text: "Testimonial text here",
    initials: "PN"
  },
  // Add more...
]
```

### To Update Doctor Info:
Edit `src/lib/config.ts`:
```typescript
CLINIC_DOCTOR_NAME="Dr. Name"
SERVICE_*_PRICE="₹Price"
// etc...
```
